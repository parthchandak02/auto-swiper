Quick start

  1. Download your platform build above.
  2. On macOS, extract the .zip and open the app. On Linux, set executable: chmod +x AutoSwiper-Linux.
  3. Open your Android emulator and launch Hinge before starting.

Prefer running from source? See the README on GitHub for setup, or run: 

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py

Customize messages

Auto-Swiper creates ~/Documents/AutoSwiper_CustomJokes.txt on first run. Edit to personalize.

  • Open via AutoSwiper --jokes
  • Lines starting with # are ignored
  • Defaults ship alongside your custom list

About

This is a desktop automation tool powered by PyAutoGUI with a modern terminal UI built using Rich. GitHub Pages hosts these static docs and download links; the actual app runs on your machine.
